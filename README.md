# 📘 Curso de JavaScript

Repositorio educativo con material organizado para aprender **JavaScript** desde lo básico hasta conceptos avanzados.  
Incluye teoría, ejemplos prácticos y ejercicios recopilados de documentación, videos y práctica personal.

---

## 📂 Contenido
1. Introducción  
2. Control de flujo  
3. Scope y hoisting  
4. Funciones  
5. Objetos  
6. Arrays y colecciones  
7. DOM  
8. Modularidad  
9. Programación asíncrona  
10. APIs  
11. Storage y persistencia  
12. Programación orientada a objetos (POO)  
13. Programación funcional  
14. Binding  
15. Manejo de errores  

---

## 🎯 Objetivo
Este curso fue creado como parte de mi formación en desarrollo full stack.  
La intención es compartir un recorrido claro y progresivo para estudiantes y desarrolladores que quieran dominar JavaScript.
